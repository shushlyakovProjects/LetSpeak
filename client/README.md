# QuizMaster
*Онлайн-сервис "QuizMaster": генератор тестов и опросников с автоматическим подсчетом баллов*

## Описание проекта:
Сервис предназначен для преподавателей, студентов и обычных пользователей, желающих создавать собственные викторины, тесты и опросники онлайн. 

Сервис позволяет пользователям быстро формировать наборы вопросов с несколькими типами ответов (текстовые поля, множественный выбор, чекбоксы), сохранять созданные тесты и проводить оценку ответов автоматически.

## Основные особенности проекта:
1. Регистрация и профиль пользователя:Пользователи регистрируются, создавая аккаунт, и сохраняют личные данные (имя, email, аватар) в общем хранилище с помощью Redux.
2. Генерация теста: Пользователям предоставляется удобный интерфейс для создания собственных тестов, включающих вопросы разного формата: одиночный выбор, множественный выбор, текстовый ответ. Данные сохраняются в базу данных и хранятся в центральном хранилище Redux.
3. Управление тестами: Интерфейс для редактирования созданных тестов, удаления и публикации на публичный доступ.
Проведение опроса:После публикации теста пользователи могут проходить его самостоятельно, заполняя форму ответов. Форма включает проверки на заполненность обязательных полей и валидацию введённых данных.
4. Оценка результатов: Система автоматически подсчитывает баллы участников, показывая итоговую статистику и предлагая советы по улучшению результата.
5. Асинхронные запросы: При создании тестов и прохождении опросов система отправляет асинхронные запросы на сервер для сохранения данных и загрузки готовых материалов.
