# LetSpeak
Цель проекта: разработка мессенджера.

## Задачи:
* Реализовать регистрацию и авторизацию пользователей ✅
* Реализовать общий чат ✅
* Возможность отправки эмодзи ✅
* Голосовые сообщения ✅
* Отправка изображений ✅
* Реализовать видеоконференции
* Реализовать личные диалоги

## Технологии:
### Frontend:
* React
### Backend:
* Node.Express
* WebSocket
* MySQL

## База данных
### Таблицы:
* users: UserId, UserName, UserLogin, UserPassword
* messages: MessageId, MessageSenderLogin, MessageSenderName, MessageContent, MessageImage, MessageVoiceContent,  MessageDate, MessageAnswerOn
